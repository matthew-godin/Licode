# licode

A confrontational coding platform.

## Setup

### Initial Setup

```
git clone https://github.com/matthew-godin/licode
cd licode
export DENO_DIR=<path-to-licode>/licode/packages
deno run --allow-net src/main.ts 
```
### To Reload Packages

```bash
deno run --allow-net --reload src/main.ts 
```
